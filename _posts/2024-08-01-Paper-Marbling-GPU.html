--- layout: post title: Paper Marbling GPU --- WebGPU Accelerated Paper Marbling.
Right click to drop paint onto canvas. The combing is done by left clicking and dragging.
For logic, checkout CPU Version. WebGPU is not yet supported on FireFox. The demo should work on Chrome.
Calculations in itself takes less than 1ms even after 8,000,000 vertices.
Just for reference, my CPU version takes more than 4000ms. More analysis to come later.
  • RenderStroke

  • Average Compute Time: 0ms

I transcribed the CPU code into WebGPU shader (WGSL) to run in WebGPU.
The bottle neck comes from mapping gpu memory into cpu memory and rendering using canvas stroke.
The concave shapes need triangulation to render, so I decided that for now, I would give rendering logic to javascript's native rendering.
